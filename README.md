# [ Fake Twitter ] Front-End React

## Requisitos

Node.js [10.16.0](https://nodejs.org/dist/v10.16.0)\
Yarn [6.4.1](https://github.com/yarnpkg/yarn/releases/tag/v1.16.0) Ou\
Npm [6.4.1](https://www.npmjs.com/package/npm/v/6.4.1)

## Passos

1. Faça o clone/download deste repositório: `git clone https://github.com/Hugofv/fake-twitter-app.git`
2. Faça o clone/download do repositório da API: `git clone https://github.com/Hugofv/fake-twitter-api.git`
3. Instale as dependências: `yarn install` ou `npm install`
4. Start a aplicação da API: `yarn start` ou `npm start`
5. Start a aplicação do Front-end: `yarn start` ou `npm start`
